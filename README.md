# 🐾 AdoptMeNow

AdoptMeNow is a web-based **pet adoption management system** with integrated animal rescue reporting, built using **ASP.NET MVC**.

The main focus of this system is to simplify and digitalize the pet adoption process while also supporting animal welfare through rescue reporting features.

---

## 📌 System Overview

AdoptMeNow is designed to manage the full **pet adoption lifecycle**, from browsing available pets to submitting and approving adoption applications.

The system also supports animal rescue reporting, allowing users to report stray or injured animals for further action by administrators.

### 🐶 Core Focus: Pet Adoption System
- Browse available pets for adoption
- Submit adoption applications
- Track adoption status (Pending, Approved, Rejected)
- View adoption history and updates
- Admin approval workflow for adoption requests

### 🆘 Supporting Feature: Animal Rescue System
- Report stray or injured animals
- Upload images and location details
- Track rescue report status (Pending, In Progress, Resolved, Rejected)
- Admin management of rescue cases

---

## 👥 User Roles

### 🧑 Adopter (User)
- Register and log in
- Browse pets available for adoption
- Submit adoption requests
- Track adoption status
- View adoption history
- Submit rescue reports
- Track rescue report progress
- Manage personal profile

---

### 🛠️ Admin
- Manage pet listings (add, update, delete)
- Review and approve/reject adoption applications
- Manage adoption workflow and status updates
- Handle rescue reports submitted by users
- Update rescue case status
- Monitor system activities
